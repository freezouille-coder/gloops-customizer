⚙️ Scripts & Checklists

Les 5 outils Maya/Python du projet + les workflows pas-a-pas.

1. 🧰 Les scripts

ScriptOu le lancerCe qu'il fait
export_traits.py Maya Script Editor 2 fonctions :
• EXPORT — Boucle Studio Library → FBX (anim/pose, bake blendshape, clean bones, FPS convert, GLB auto-convert)
• BATCH EXPORT PROPS — Selection de meshes → 1 FBX par mesh (skeleton + mesh, pas d'anim)
build_props_manifest.py Terminal Python Scanne fbx/ANIM/*/PROPS/*.fbx, met a jour pairedProps dans manifest.json.
⚠️ A relancer apres chaque ajout/suppression de prop FBX.
fbx_to_glb.py Terminal Python (ou via Blender headless) Batch convert FBX → GLB via Blender. Options : --compress (Draco), --no-anim, --scale.
GUI Tkinter si lance directement en Python.
export_trees.py Maya Script Editor Export MASH/transforms selectionnes → JSON {x, y, z, ry, s} par instance.
Pour le placement d'arbres via game-tree-instancer.js.
BindCleaner.py Maya Script Editor Nettoie les bindPose en double dans la scene. Fix les warnings d'export.

2. ✅ Ajouter une nouvelle animation

  1. Creer le .anim ou .pose dans Studio Library
  2. Lancer export_traits.py → EXPORT
    • Source = dossier Studio Library
    • Target = fbx/ANIM/<Category>
    • Selectionner Skeleton Root + Geometries
  3. Le FBX apparait dans le dossier
  4. Relancer build_manifest.py si le manifest n'est pas auto-updated
  5. Refresh navigateur → la nouvelle anim apparait dans le dropdown

3. ✅ Ajouter un prop paire (lunettes, chapeau...)

  1. Dans Maya, nommer le mesh exactement comme le FBX d'animation correspondant (ex: Hat_CowBoy)
  2. Selectionner le(s) mesh(es) dans le viewport
  3. Dans export_traits.py, set Target = fbx/ANIM/<Category>/PROPS/
  4. Cliquer BATCH EXPORT PROPS
  5. Lancer python scripts/build_props_manifest.py
  6. Refresh → le prop se charge automatiquement quand on selectionne l'animation
  7. Si offset necessaire → ajuster dans le panneau Props (sliders live) → bouton 📋 Copy JSON → coller dans config/paired-offsets.json

4. ✅ Ajouter des textures pour un nouveau materiau

  1. Creer le dossier images/chr/<matName>/
  2. Nommer les textures avec la convention <matName>_Suffixe.png (voir doc Textures)
  3. Pour les variantes : <matName>_Pattern.N.png (N = 0, 1, 2...)
  4. Relancer build_manifest.py
  5. Refresh → les textures apparaissent dans le panneau MAT

5. ✅ Preparer les zones de la map (Maya)

  1. Creer des planes/polygones nommes zone_<type>_<nom> (voir doc Zones)
  2. Les poser a meme le sol de l'ile
  3. Les grouper sous un group ZONES
  4. (Optionnel) NURBS curves path_drive_right, path_drive_left
  5. Exporter en FBX separe → fbx/WORLD/zones.fbx
  6. Au runtime, le code les parse par nom et les rend invisibles

6. ✅ Batch convert FBX → GLB (optimisation)

GLB charge ~3x plus vite que FBX et pese ~70% de moins avec Draco compression.

  1. Option A : cocher "Auto-convert FBX → GLB" dans export_traits.py avant l'export
    • Les FBX sont convertis en GLB dans fbx/glb/<Category>/ immediatement apres l'export
  2. Option B : lancer manuellement python scripts/fbx_to_glb.py
    • GUI Tkinter avec auto-detect Blender, input/output folders, options Draco
    • Ou en CLI : python scripts/fbx_to_glb.py --input ... --output ... --compress

7. 🐛 Troubleshooting

ProblemeCause probableFix
Paired props n'apparaissent pas manifest.json sans pairedProps (sync Google Drive l'a ecrase) python scripts/build_props_manifest.py
Prop mal oriente Type C exporte avec une pose different de la T-pose Offset rotation dans paired-offsets.json — OU reset skeleton T-pose avant export Maya
NPC avec les memes lunettes que le player Clone SkeletonUtils inclut les props attaches Deja fixe — les paired props sont tagges et strip au clone
"No module named 'studiolibrarymaya'" Module pas charge dans la session Maya Ouvrir Studio Library une fois avant de lancer le script
BindCleaner fails Syntax Python 2 vs 3 Deja fixe (utilise print())
Shader d'un prop n'apparait pas dans MAT Multi-material par face (array), mesh sans .skinning flag Deja gere — chaque slot de l'array devient sa propre entry dans le pool
